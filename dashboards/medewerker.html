Centrale vraag
Zetten we de juiste mensen op de juiste projecten — en houden ze dit vol?
Bronnen
Keeping (uren, indicatief). Facturabel/niet-facturabel is een werkafspraak/label en kan per team verschillen.
Inzichten
  • Inzet per project (gestapeld) → versnippering vs focus + druk per rol.
  • Facturabel vs niet-facturabel → productiviteit vs overhead.
Kijkpunt: belasting >100% of veel context-switch → risico op kwaliteitsverlies en vertraging. Bij dalende facturabiliteit: check overleg/interrupties en herverdeel werk.
Prototype: statische HTML + Tailwind CDN. Data is mock.
Volgende stap: Blade components + Livewire state + echte API-koppeling.
